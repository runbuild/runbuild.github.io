London Stage Database Project

Tech Stack:
  • HTML/CSS
  • JS
  • PHP
  • Python
  • MySQL
  • PS
  • Node.js

Completed as part of a grant project through USU, the goal was to revive a digital database from the 1970s of The London Stage (1660-1800 London plays). The website needed to provide a powerful search interface to the revived database, while showing the various stages the data went through in the process. The goal was to maintain a simple interface for the average user while providing more powerful options for the seasoned researcher. Due to limited funding, many time-intensive features (like speedy searches) will take place through future grants.
